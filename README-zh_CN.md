## 可用的脚本

在项目的根目录，你可以运行以下脚本：


### `yarn install`

安装项目依赖

### `yarn start`

在开发模式下运行应用<br />

网站的默认地址为：localhost:3000

网页代码渲染地址为：http://localhost:3000/#/code

### `yarn test`

运行测试

### `yarn build`

编译项目，生成在 `build` 文件夹.<br />
